# Простейшая сложная нейросеть
по мотивам задачи из курса по ML

Программа, которая вычисляет матричную функцию и её производную по заданному графу вычислений.

#### зачем?
Если вы пишете свою очень быструю реализацию базовых слоев нейронных сетей и вычисления производных слоев, то можно иcпользовать эту наивную и прозрачную "реализацию" для стресс-тестов своей реализации. 

#### Как пользоваться
В первой строке содержится три целых положительных числа N, M, K — число вершин в графе вычислений, число входных параметров (вершин) и число выходных параметров (вершин). Далее следует N строк — описание вершин графа вычислений. i-я из этих строк содержит описание i-й вершины:
- `var r c` — входной параметр функции, матрица состоящая из r строк и c столбцов.
- `tnh x` (1≤x<i) — матрица из значений гиперболического тангенса вычисленного от соответствующих компонент матрицы, полученной из x-й вершины графа вычислений.
- `rlu α−1 x` (1≤x<i) — матрица из значений функции параметрического линейного выпрямителя с параметром α, вычисленной от соответствующих компонент матрицы полученной из x-й вершины графа вычислений. α−1 — целое число. Производная в нуле равна единице.
- `mul a b` (1≤a,b<i) — произведение матриц, полученных из a-й b-й вершины графа вычислений соответственно.
- `sum len u1 u2 ... ulen` (∀1≤j≤len:1≤uj<i) — сумма матриц, полученных из вершин u1,u2,…,ulen графа вычислений.
- `had len u1 u2 ... ulen` (∀1≤j≤len:1≤uj<i) — произведение Адамара (покомпонентное) матриц, полученных из вершин u1,u2,…,ulen графа вычислений. 

Далее следует описание M матриц — входных параметров соответствующих вершин графа вычислений в порядке возрастания их индексов.
Затем следует описание K матриц — производных функции по соответствующим выходным вершинам в порядке возрастания их индексов.

#### Выходные данные

Выведится K матриц — значение параметров соответствующих выходных вершин графа вычисления в порядке возрастания их индексов. Затем выведтся M матриц — производных функции по соответствующим входным вершинам в порядке возрастания их индексов.

#### Пример

на вход:
```
6 3 1
var 1 3
var 3 2
var 1 2
mul 1 2
sum 2 4 3
rlu 10 5
-2 3 5
4 2
-2 0
2 1
4 -2
-1 1
```

вывод:
```
0.0000000000 -0.1000000000
-3.8000000000 2.0000000000 -1.9000000000
2.0000000000 -0.2000000000
-3.0000000000 0.3000000000
-5.0000000000 0.5000000000
-1.0000000000 0.1000000000
```
